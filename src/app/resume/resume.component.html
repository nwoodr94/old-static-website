This Site


I built this website in Angular, and deployed it on an S3 instance. After buying this domain, I passed the S3 name servers to my provider. Route 53 handles all my dns resolution, and I enlisted CloudFront to pass my ACM certificate to require HTTPS protocol.

The code for this site is hosted on my GitHub. True to the culture of DevOps, I can make any changes I want, and immediately redeploy the site from the command line.
